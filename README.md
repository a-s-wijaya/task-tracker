# Task Tracker

This project is a simple task tracker app built with React. It uses JSON-server as a mock backend.

This project provided by [Traversy Media crash course](https://www.youtube.com/watch?v=w7ejDZ8SWv8).

## Usage

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```

### To build for production

```
npm run build
```
